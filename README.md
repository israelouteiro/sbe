# YellowPages

- to up and run backend

```sh
    $ cd yellowpages_api && yarn && yarn start
```

- to up and run frontend

```sh
    $ cd yellowpages && yarn && yarn start
```

- to run frontend test with cypress

```sh
    $ cd yellowpages && yarn test
```
 
- to up and with docker

```sh
    $ docker-compose up
```
